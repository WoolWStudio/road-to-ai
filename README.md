# 12 周高强度 AI 求职冲刺计划

欢迎来到你的 AI 工程师转型之旅！本项目包含了 12 周高强度求职冲刺计划的所有代码和学习产出。

## 学习路线图

| 阶段 🎯 | 时间安排 | 核心聚焦点 | 关键产出 (作为面试作品集) 💼 |
| :--- | :--- | :--- | :--- |
| **第一阶段：快速应用层** | 第 1-3 周 | API 基础、Prompt 工程、流式输出 (Streaming) | **项目一：定制化 AI 写作助手** (纯前端/Node.js 实现，最快建立信心) |
| **第二阶段：数据与核心逻辑** | 第 4-7 周 | 向量数据库、RAG 工作流、Tool Use 工具调用 | **项目二：支持文档解析的知识库系统** (展示处理非结构化数据的能力) |
| **第三阶段：高阶工程与护城河** | 第 8-10 周 | 引入 Python (FastAPI)、Evals 评估体系、可观测性 (Tracing) | **项目三：多 Agent 研究助手** (展示后端重构与系统稳定性保障能力) |
| **第四阶段：商业化与求职** | 第 11-12 周 | 多模态集成、AI 系统设计 (System Design) 面试准备 | **项目四：垂直领域完整 SaaS** (整合前三个项目，加入用户认证与计费) |

### 核心策略
1. **扬长避短：** 前 7 周完全在前端舒适区（TypeScript + Next.js + Vercel AI SDK）狂奔，快速积累 2 个完整的 AI 落地项目。
2. **精准补强 Python：** 在第 8 周需要做评估（Evals）和重构后端微服务时，直接在实战中用 FastAPI 现学现卖，不浪费时间单独学语法。
3. **直击高薪痛点：** 重点掌握 **Evals（评估体系）** 和 **可观测性**。在面试中展示“系统化测试 LLM 准确率”及保障系统稳定性的能力，实现差异化竞争。

## 目录结构
- `phase1-ai-writing-assistant/`: 定制化 AI 写作助手 (第 1-3 周)
- `phase2-rag-knowledge-base/`: 支持文档解析的知识库系统 (第 4-7 周)
- `phase3-multi-agent-research/`: 多 Agent 研究助手 (第 8-10 周)
- `phase4-vertical-saas/`: 垂直领域完整 SaaS (第 11-12 周)

> 💡 **提示：** 每周预计投入 15-20 小时的高质量时间。保持专注，代码见！
